Generating a json schema from a json file

Install a tool that can convert a json file to a schema:
1) sudo npm install -g generate-schema
2) Use it by executing "generate-schema myjsonfile.json > myjsonschema.json"

Alternatively:
1) Execute: "pip install genson" in the terminal
2) Execute: "genson myjsonfile.json > schema2.json" in the terminal
